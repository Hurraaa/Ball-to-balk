# NEON BREAKER

Premium görünümlü, pseudo-3D efektli tuğla kırma oyunu. HTML5 Canvas + saf JavaScript.

## Oyna

`index.html` dosyasını tarayıcıda aç. Hiçbir bağımlılık yok.

## Kontroller

- **← / →** veya **Fare / Dokunmatik**: Raketi hareket ettir
- **Boşluk** veya **Tık**: Topu fırlat
- **P**: Duraklat / devam et

## Özellikler

- Bevel ve gölge ile pseudo-3D tuğlalar
- Glow efektli top ve raket, hareket izi (trail)
- Patlama parçacık efektleri ve skor popup'ları
- 4 farklı power-up: Geniş raket, çoklu top, yavaş top, ekstra can
- Combo sistemi — ardışık tuğla kırma bonus puan verir
- Artan zorlukta seviyeler
- Web Audio API ile prosedürel ses efektleri
- Animasyonlu perspektif zemin ve neon görünümlü UI
